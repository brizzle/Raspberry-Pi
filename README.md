## Setup

Step 2. If the IP is known then type into the terminal

```bash
$ ssh 192.168.1.5 -l pi
```

else then you will need to physically go to the terminal on the Raspberry Pi to get the IP Address. From the command line enter:

```bash
$ sudo ifconfig
```

Look for "inet addr".
